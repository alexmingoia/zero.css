ZeroCSS

  • Beautiful, functional styling without classes.
  • Elements for layout and grid combinations.
  • Attributes for spacing and alignment.

Usage

Include links to the reset and zero.css:

<link href="https://cdn.jsdelivr.net/gh/sindresorhus/modern-normalize@main/modern-normalize.css" type="text/css" rel="stylesheet" />
<link href="https://cdn.jsdelivr.net/gh/alexmingoia/zero.css@main/zero.css" type="text/css" rel="stylesheet" />

Elements

Sticky <footer>

If a footer is a direct descendent of body, and there is also an adjacent main and/or header element, the footer will stick to the bottom of the page.

Attributes

Alignment

  • align="horizontal|vertical" Aligns content along horizontal or vertical axis.
  • horizontal="left|center|right" Controls horizontal alignment.
  • vertical="top|center|bottom" Controls vertical alignment.

Text

  • text-align="left|center|right"
  • text-size="xs|sm|md|lg|xl|2xl|3xl|4xl"

Spacing

  • max-width="sm|md|lg|xl"
  • padding="xs|sm|md|lg|xl"
  • padding-x="xs|sm|md|lg|xl"
  • padding-y="xs|sm|md|lg|xl"
  • padding-top="xs|sm|md|lg|xl"
  • padding-left="xs|sm|md|lg|xl"
  • padding-right="xs|sm|md|lg|xl"
  • padding-bottom="xs|sm|md|lg|xl"
